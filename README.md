# KeePassHTTPClient

## Requirements
 - php >= 5.5
 - [guzzlehttp/guzzle](https://github.com/guzzle/guzzle) >= 6
 
## Install

Best way is to install with [composer](https://getcomposer.org/)

```sh
$ composer require valicek1/keepasshttpclient
```
